## üì• Installation Guide

1.  **Download:** Get the latest `.zip` file from the [Releases Page](https://github.com/jordanx09-web/YT-Music-Morphe/releases).
2.  **Flash:** Open **Magisk** or **KernelSU**, go to the *Modules* tab, and tap *Install from Storage*.
3.  **Select:** Choose the `YT-Music.Morphe-v...zip` file.
4.  **Reboot:** Restart your device to apply the changes.

> **üí° Pro Tip:** Ensure you have the official YouTube Music app installed (or detached) depending on your mounting preference.
